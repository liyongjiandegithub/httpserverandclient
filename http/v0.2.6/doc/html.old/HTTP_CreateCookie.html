HTTPCookieInfo* HTTP_CreateCookie

( HTTPCookieContext* ctx,
  const HTTP_CHAR* name,
  const HTTP_CHAR* value,
  const HTTP_CHAR* domain,
  const HTTP_CHAR* path,
  RTP_TIMESTAMP* expires )

Create a new cookie for a cookie database.

Documentation

Creates a new cookie. This cookie can be manipulated freely by HTTP_CookieSetName, HTTP_CookieSetName, etc., until it is inserted back into the database by HTTP_ReleaseCookie.

The newly created cookie will not be sent to any servers until it is released using HTTP_ReleaseCookie.

All strings passed into this function will be copied into buffers controlled by the cookie context.

Parameters:
ctx - the cookie database to create the cookie for
name - the cookie name
value - the cookie value
domain - the domain to send this cookie to
path - the path prefix of request targets to send this cookie to
expires - the expiration time of the cookie
Returns:
pointer to cookie if found, 0 otherwise
See Also:
HTTP_ReleaseCookie
