void HTTP_SetCookieFilter

( HTTPCookieContext* ctx,
  unsigned (*filterFn) ( void* filterData,
  HTTPCookieContext* ctx,
  HTTPCookieInfo* cookie ),
  void* filterData )

Set a function for a given cookie context to filter incoming cookies.

Documentation

Provides a way to select which cookies are accepted and stored locally by the HTTP client. All incoming cookies will be given to the filter function; if it returns 0, the cookie is rejected and not stored; otherwise, the cookie is accepted and stored in the local database.

Parameters:
ctx - cookie context to set filter for
cookie - filter function
filterData - opaque data passed to the filter function to provide context for the filter
Returns:
0 on success, negative on failure
See Also:
HTTP_SetCookie
