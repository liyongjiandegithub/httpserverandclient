int HTTP_InitCookieContextMT

( HTTPCookieContext* ctx,
  HTTP_INT32 maxCookieBytes,
  RTP_MUTEX lock )

Initialize a thread-safe cookie context.

Documentation

Does the same as HTTP_InitCookieContext, but also enables threadsafe operation. The HTTP client cookie module must be compiled with the preprocessor symbol HTTP_CLIENT_COOKIE_THREADSAFE defined (which enables thread-safe mode) to use this function.

Parameters:
ctx - uninitialized struct to hold cookie data
maxCookieBytes - size limit on cookie storage
lock - a mutex handle created by rtp_sig_alloc_mutex; the mutex must be seperately destroyed after HTTP_DestroyCookieContext is called, and must not be used for any other purpose while the cookie context is valid.
Returns:
0 on success, negative on failure
See Also:
HTTP_InitCookieContextMT, HTTP_DestroyCookieContext
