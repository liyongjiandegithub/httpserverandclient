int HTTP_SetCookieUC

( HTTPCookieContext* ctx,
  const HTTP_UINT16* cookieStr,
  const HTTP_UINT16* defaultDomain,
  const HTTP_UINT16* defaultPath )

Set a cookie in 16-bit Unicode format.

Documentation

Performs the same function as HTTP_SetCookie, except that the strings passed are 16-bit Unicode instead of 8-bit ASCII.

Parameters:
ctx - the cookie database in which to store the cookie if it is accepted
cookieStr - the contents of the HTTP "Set-Cookie" option header
defaultDomain - the host name that the original request was sent to
defaultPath - the path requested on the server
Returns:
0 on success, negative on failure
See Also:
HTTP_SetCookie
