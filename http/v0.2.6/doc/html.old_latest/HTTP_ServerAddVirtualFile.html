int HTTP_ServerAddVirtualFile

(HTTPServerContext* server,
  const HTTP_CHAR* path,
  const HTTP_UINT8* data,
  HTTP_INT32 size,
  const HTTP_CHAR* contentType)

Assign a memory array to be returned when the URL is requested.

Documentation

With this function you can assign a memory array to be returned when the URL is retrieved by a client.

Parameters:
server - Pointer to structure that was initialized by HTTP_ServerInit
path - The URL
data - The content to return when GET is performed on the URL.
size - Size of the content
Returns:
0 on success, negative on failure
