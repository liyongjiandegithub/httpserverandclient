int HTTP_ServerInit

(HTTPServerContext* server,
  const HTTP_CHAR* name,
  const HTTP_CHAR* rootDir,
  const HTTP_CHAR* defaultFile,
  HTTP_INT8 httpMajorVersion,
  HTTP_INT8 httpMinorVersion,
  HTTP_UINT8* ipAddr, HTTP_INT16 port,
  HTTP_INT16 ipType,
  HTTP_BOOL allowKeepAlive,
  HTTPServerConnection* connectCtxArray,
  HTTP_INT16 maxConnections,
  HTTP_INT16 maxHelperThreads )

Initialize an HTTP Managed Server instance.

Documentation

This function must be called once before any other managed server calls to create a context in which the server will operate.

Parameters:
server - pointer to uninitialized HTTPServerContext struct
name - Server name. Sent in HTTP header.
rootDir - Path to the root directory for content.
defaultFile - Default Filename if "\" or "path\" is passed with no file name.
httpMajorVersion - Normally 1 for HTTP 1.1
httpMinorVersion - Normally 1 for HTTP 1.1
ipAddr - V4/V6 examples: ipaddr[4] = {192, 168, 0, 6}; ipAddr = "fe80::20b:dbff:fe2f:c162";
port - port number (80) usually
ipType - ipversion 4 or 6
allowKeepAlive - If HTTP version < 1.1 enables keep-alive support. For 1,1 and greater keep-alive is the default.
connectCtxArray - pointer to uninitialized bytes, must be (sizeof(HTTPServerConnection) * maxConnections)) bytes.
maxConnections - The maximum limit on simultaneous HTTP server connections
maxHelperThreads - If HTTP_SERVER_MULTITHREAD is defined, the max number of helper threads to spawn
Returns:
0 on success, negative on failure
See Also:
HTTP_ServerDestroy
