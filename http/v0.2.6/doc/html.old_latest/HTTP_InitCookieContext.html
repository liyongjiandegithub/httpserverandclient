int HTTP_InitCookieContext

( HTTPCookieContext* ctx,
  HTTP_INT32 maxCookieBytes )

Initialize a cookie database context.

Documentation

Initilizes a HTTPCookieContext struct, which holds cookie information for a HTTP client. Old cookies will be discarded automatically to ensure that the total memory usage of the HTTPCookieContext never exceeds maxCookieBytes.

Parameters:
ctx - uninitialized struct to hold cookie data
maxCookieBytes - size limit on cookie storage
Returns:
0 on success, negative on failure
See Also:
HTTP_InitCookieContextMT, HTTP_DestroyCookieContext
