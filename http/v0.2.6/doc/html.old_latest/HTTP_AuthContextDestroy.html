void HTTP_AuthContextDestroy

( HTTPAuthContext* ctx )

Destroy an HTTPAuthContext

Documentation

This function will free all data stored in the credential database for this HTTPAuthContext. It should only be called after any HTTPManagedClient's which use it have been destroyed using HTTP_ManagedClientDestroy.

Parameters:
ctx - The HTTP Client Authentication context to destroy
Returns:
0 on success, negative on failure
See Also:
HTTP_AuthContextInit
