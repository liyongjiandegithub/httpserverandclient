Advanced Web server example

Starts a web server and demonstrate implementing custom request handlers.
This code demonstrates the necessary steps to start an advanced web server application.

  • How to display dynamically inserted web pages from a virtual file table.
  • How to assign callback handlers to handle specific Urls.
  • How to process form data sent by a POST.
  • How to save uploaded file data.
  • How to use Ajax to dynamically update HTML elements from dynamic server content.

          The advanced web server example is still under development. Inspect the examples directory for the lastest source code
