int HTTP_SetCookie

( HTTPCookieContext* ctx,
  const HTTP_CHAR* cookieStr,
  const HTTP_CHAR* defaultDomain,
  const HTTP_CHAR* defaultPath )

Enter a cookie into the database.

Documentation

This function is normally called automatically by the HTTP client when a server response contains a "Set-Cookie" header option. It will pass the cookie through the user-specified filter function, if it has been set.

In case the cookie does not explicitly specify a DNS domain or server path for which the cookie applies, the host name that the original request was sent to, along with the path on that server that the request was for, should be passed into this function as the defaultDomain and defaultPath parameters.

Parameters:
ctx - the cookie database in which to store the cookie if it is accepted
cookieStr - the contents of the HTTP "Set-Cookie" option header
defaultDomain - the host name that the original request was sent to
defaultPath - the path requested on the server
Returns:
0 on success, negative on failure
See Also:
HTTP_SetCookieFilter
