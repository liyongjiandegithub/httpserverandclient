int HTTP_ReleaseCookie

( HTTPCookieContext* ctx,
  HTTPCookieInfo* cookie )

Give a cookie back to the cookie database.

Documentation

Inserts a cookie into the given cookie database. The cookie is created by HTTP_CreateCookie or obtained using HTTP_FindMatchingCookie or HTTP_FindExactCookie.

Returns:
0 on success, negative on failure
See Also:
HTTP_CreateCookie, HTTP_FindMatchingCookie, HTTP_FindExactCookie
