int HTTP_SetAuthorization

( HTTPAuthContext* ctx,
  HTTP_CHAR* host, HTTP_UINT16 port,
  HTTP_CHAR* path,
  HTTP_CHAR* userName,
  HTTP_CHAR* password,
  HTTP_INT32 ttlSec )

Create or update credentials for a set of URLs.

Documentation

This function sets the user name and password to be used to access all URLs beginning with the given path string on the given server. If there are no credentials already in the database for the given server/path, then a new entry is created with the given user/password using Basic authentication method. If the server/path is already in the database, the user name and password used for access on that server/path are updated and the authentication method is left unchanged.

Only one user (with password) is stored per server/path.

Parameters:
ctx - authorization database context
host - host name to set credentials for
port - server port number
path - the credentials are set for all paths on the given server that begin with this path string
userName - the user name to use when sending requests to the given server/path
password - the password to associate with the user name
ttlSec - how long to use this authorization before discarding it from the database
Returns:
0 on success, negative on failure
See Also:
HTTP_AuthContextInit
