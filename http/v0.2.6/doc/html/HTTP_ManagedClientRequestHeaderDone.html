int HTTP_ManagedClientRequestHeaderDone

( HTTPManagedClientSession* session,
  HTTPRequest* request )

Send CR LF to terminate HTTP request header and update session state. (low level).

Documentation

HTTP_ManagedClientRequestHeaderDone is provided for implementing http based protocols such as SOAP, SSDP and GENA. Use caution when calling this function. Most application needs can be satisfied by using one of the higher level functions HTTP_ManagedClientGet, HTTP_ManagedClientPut, or HTTP_ManagedClientPost.

Returns:
0 on success, negative on failure
